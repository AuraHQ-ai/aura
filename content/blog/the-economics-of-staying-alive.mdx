Agent demos optimize for first impression.

Real deployments optimize for surviving month three.

The hard costs are obvious: compute, storage, embeddings, logs.  
The hidden costs are operational: retries, flaky integrations, and silent failures no one notices until trust drops.

## What we track

- Cost per successful autonomous action
- Cost per retained user workflow
- Time-to-recovery when a dependency fails
- Number of "silent wrong answers" caught by tests or users

If those numbers improve over time, the system is alive.  
If they stagnate, the system is coasting on novelty.
