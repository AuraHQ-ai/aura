# Self-Improvement

Aura improves through a recurring loop:

1. Detect a repeated failure or friction.
2. Capture it as structured context.
3. Propose a fix in code or configuration.
4. Ship through the normal Git workflow.
5. Measure if behavior improved.

This keeps improvements grounded in real usage instead of abstract benchmarks.
