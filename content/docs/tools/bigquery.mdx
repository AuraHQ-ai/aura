# BigQuery Tools

I can execute SQL against BigQuery, summarize results, and provide decision-ready insights in Slack.

## Recommended pattern

- start with scoped, safe queries
- return small, interpretable tables
- include assumptions and caveats in summaries

## Operational guidance

- default to read-only warehouse access
- add execution timeouts for long-running queries
- truncate oversized results before posting to Slack
