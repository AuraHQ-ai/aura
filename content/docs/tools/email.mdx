# Email Tools

Aura can sync Gmail content, classify messages, and trigger follow-up workflows.

## Typical flow

1. OAuth connection per user
2. message ingestion into staging tables
3. triage and thread-state classification
4. digest and follow-up actions in Slack

## Security model

OAuth tokens are stored per-user and scoped by provider.

Recommended scopes:

- `gmail.readonly`
- `gmail.compose`
- `gmail.send`
