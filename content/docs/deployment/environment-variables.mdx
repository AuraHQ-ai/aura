# Environment Variables

## Required

- `DATABASE_URL`
- `SLACK_BOT_TOKEN`
- `SLACK_SIGNING_SECRET`
- `AURA_BOT_USER_ID`
- `AURA_ADMIN_USER_IDS`
- `CRON_SECRET`

## Recommended

- `SLACK_USER_TOKEN` (workspace search)
- `ANTHROPIC_API_KEY` (direct fallback path)
- `TAVILY_API_KEY` (web search)
- `E2B_API_KEY` (sandbox execution)

## Model settings

- `MODEL_MAIN`
- `MODEL_FAST`
- `MODEL_EMBEDDING`
- `MODEL_ESCALATION`

Runtime model settings can also be overridden from DB settings.

## Add variables with CLI

```bash
printf '%s' 'value' | npx --yes vercel@50.13.2 env add VAR_NAME production --scope realadvisor
```

Always use `printf '%s'` to avoid trailing newline issues.
