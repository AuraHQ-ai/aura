# Environment Variables

Core runtime variables:

- `DATABASE_URL`
- `SLACK_BOT_TOKEN`
- `SLACK_SIGNING_SECRET`
- `AURA_BOT_USER_ID`
- `AURA_ADMIN_USER_IDS`
- `CRON_SECRET`

Optional but useful:

- `SLACK_USER_TOKEN`
- `TAVILY_API_KEY`
- `ANTHROPIC_API_KEY`
