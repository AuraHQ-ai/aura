# One-Click Deploy

Use the hosted deployment button from the repository README, then:

1. set required environment variables,
2. configure Slack event + interaction URLs,
3. reinstall the Slack app,
4. verify `/api/health` and cron auth.

## After deploy checklist

- Validate Slack event signatures are accepted
- Confirm heartbeat cron can run with `CRON_SECRET`
- Send one DM and one channel mention test
- Verify memory persistence in Postgres
